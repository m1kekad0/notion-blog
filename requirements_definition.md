# 要件定義書

## 1. アプリケーション概要
**Notion Blog (仮)**
Notion を CMS（コンテンツ管理システム）として利用し、Notion で作成した記事を Next.js で構築したフロントエンドで表示するブログアプリケーション。

## 2. 目的・背景
*   Notion の使いやすさを活かして記事執筆・管理を行いたい。
*   SEO やカスタマイズ性の高いブログを構築したい。

## 3. ターゲットユーザー
*（誰に使ってもらいたいか）*

## 4. 機能要件
### ユーザー向け機能
*   **記事一覧表示:**
    *   カード型レイアウト（アイキャッチ画像 ＋ タイトル ＋ 概要 ＋ タグ/日付）
    *   ページネーションまたは無限スクロール
*   **記事詳細表示:**
    *   Notion のブロック内容を忠実に再現（画像、引用、コードブロック等）
    *   **Mermaid 記法によるダイアグラム表示対応**
    *   目次（TOC）の自動生成と表示
*   **記事の分類・検索:**
    *   タグ別一覧表示
    *   カテゴリー別一覧表示
    *   キーワード検索機能
*   **UI/UX:**
    *   ダークモード切替機能
*   **インタラクション:**
    *   コメント投稿機能（Giscus を採用）

### 管理者向け機能
*   **記事管理:** Notion 上で行う（執筆、公開ステータス変更、タグ/カテゴリー設定）

## 6. 技術スタック（確定）
*   **Frontend Framework:** Next.js (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **CMS:** Notion API
*   **Deployment:** Vercel
*   **Comment System:** Giscus (GitHub Discussions 連携)
*   **Diagrams:** Mermaid.js (via react-mermaid etc.)
